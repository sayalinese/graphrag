### final 关键字：

- 类变量的类型需要首字母大写
- 如果是传统的int,boolen就不需要大写了
- 一旦final后就会成为常量无法被修改

```java
public class FinalVar{
    public static void main(String[] args){
        final String STRING="shiyanlou";
        // STRING="hhh";
        // 常量final无法被修改
        System.out.println(STRING);
    }
}
```

### String：

1. #### 地址对应关系：

- 相同字符串的变量指向相同地址
- new会新开辟内存

```java
public class compear {
    public static void main(String[] args){
        String s1 = "abc";
        String s2 = new String("abc");
        boolean b = (s1 == s2);
        System.out.println(b);
		//false
        
        String s3 = "abc";
        String s4 = "abc";
        boolean c = (s3 == s4);
        System.out.println(c);
		//true
    }
}
```

2. #### 字符串比较

- `equals()` ：正常比较

- `equalsIgnoreCase()`：忽略大小写比较

- `==`比较内存地址

- `.length()`求长度

  ```java
  public class StringTest {
      public static void main(String[] args){
          String s = new String("Java");
          String m = "java";
          System.out.println("用equals()比较，java和Java结果为"+s.equals(m));
          // false
          
          System.out.println("用equalsIgnoreCase()比较，java和Java结果为"+s.equalsIgnoreCase(m));
          // true
          
      }
  }
  ```

3. #### 字符串连接：

- `+`

- `concat()`

  ```java
  String s0 = new String("Hello ");
  String s1 = "World" + "!";   //+号连接
  String s2 = s0.concat(s1); //concat()方法连接
  System.out.println(s2);

  ```

4. #### 字符串提取：

- `charAt() `根据索引取字符

  ```java
  String s = "abc";
  char c = s.charAt(1);
  // b
  ```

- 一堆提取方法

  ```java
  public class StringTest {
      public static void main(String[] args) {
           String s = "abcdefabc";
           System.out.println("字符a第一次出现的位置为"+s.indexOf('a'));
           System.out.println("字符串bc第一次出现的位置为"+s.indexOf("bc"));
           System.out.println("字符a最后一次出现的位置为"+s.lastIndexOf('a'));
           System.out.println("从位置3开始到结束的字符串"+s.substring(3));
           System.out.println("从位置3开始到6之间的字符串"+s.substring(3,6));
      }
  }
  ```

5. 修改字符串的方法

- ` StringBuilder`字符串容器（不限数据类型）

  - 默认容量为16
  - StringBuilder("字符")会变为16+字符.length()

- `.capacity()`查看容器的大小

- `.toString()` 对类数据类型修改

  ```java
  StringBuilder s1 = new StringBuilder();
  s1.append("java");
  StringBuilder s2 = new StringBuilder(5);
  StringBuilder s3 = new StringBuilder("shiyanlou");
  System.out.println("s1:" + s1.toString() + "\tcap:" + s1.capacity());
  System.out.println("s2:" + s2.toString() + "\tcap:" + s2.capacity());
  System.out.println("s3:" + s3.toString() + "\tcap:" + s3.capacity());
      }
  }
  
  <<< 
  s1:java cap:16
  s2:     cap:5
  s3:shiyanlou    cap:25
  ```

### 位运算

1. &	同1为1，否则为0
2. 丨       同0为0，否则为1
3. ^         对应位相同位0，否则为1
4. ~          反码
5. <<       左移一位
6. \>>       右移一位
7. \>>>     补码，同上，但移完补0（对负数有用）
