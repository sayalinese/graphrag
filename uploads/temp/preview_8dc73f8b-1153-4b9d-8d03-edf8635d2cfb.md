# DrissionPage

## 特征：

- 类Selenium的web自动化工具
- 没有webdriver特征，相比于Selenium不易被反爬拦截
- 绕开JS逆向，解析ajax动态渲染网页
- 通过可以通过控制浏览器操作抓取懒加载页面

## 基础用法

1. 初始化：`打开指定网页`

   ```python
   page = ChromiumPage()
   url = 'https://www.goofish.com/'
   page.get(url)
   ```

2. 选定文本框：`类定位，id定位，xpath定位`

   ```py
   page('***').input('显卡坞')
   ```

3. 点击：

   ```py
   page('***').click()
   ```
   > 注意,<div>标签作为触发项目时可能因为调用的js或css，要用xpath才能执行

4. 等待：

   - `page.wait.load_start()`等待加载成功
   - `page.wait(时间)`等待加载时间（但加载成功就跳过）

5. 侦听api：

   ```py
   page.listen.start("")
   ```

   > 其中侦听api通过找到评论的https包的表头里面找![image-20250401154328102](https://1696013933.oss-cn-chengdu.aliyuncs.com/image/202504011543774.png)
   >
   > - 且侦听后会一直侦听这个api的包
   > - 同类型的包api相同
   > - 这个包要在你加载界面后才会出现,或者刷新界面才刷新
   > - 每包有时候需要战术`page.refresh()`刷新

6. 提取json:

   - 返回了上图的那个响应`(response)`
   
   
      - `sup = page.listen.wait()`
   
   
   
      - `sub = sup.response.body`
   

   > 接下来是枯燥且乏味的观察json并提取的过程......

7. 处理翻页问题：

   找到对应翻页按键:用xpath才能识别`<div>`,如果是`<buttom><search>`这些就随意

   <button type="button">点击我！</button>

     <form>
       <input name="srchw3" id="srchw3" placeholder="像这种">
     </form>
   
8. 滚动界面:

   - `page.scroll.to_bottom()`到当前最底部
   - page.scroll.top/bottom/left/right(距离)

# 思路



### 1. **通过发掘闲鱼商家的文案与评论判定商家是否为贩子或骗子**

贩子：

- 清一色完美文案，完美图片。讨好客户，喜欢在商品上动小手脚来隐瞒瑕疵（贴个啥，然后撕开后不退货，但恰好在撕开后才能看到瑕疵）
- 一分钱一分货，会自动根据商品成色调整价格
- 可能特征：头像和简介表面自己是女生（女大一周自用），或者什么什么店铺
- 文案重复率高
- 通常信用极好（容易被忽略，因为交易做多了，缺斤少两不会扣信用）
- 功能描写过于详细，几几新
- 图片怕得过于正规，方向视角很正

骗子：

- 低于市场价的产品，完美展示文案，要求尽快收货（不包邮或表面不退货）。走量文案多种商品同一套路，小白杀手，老鱼一眼能辨真假

个人买家：

- 通常愿意表面瑕疵，脾气一般，商品态度爱答不理，文案简单。发售价格稍低于市场的个人买家的货会被贩子秒，所以不存在价格较低的优势。

- 有捡漏可能，不太会会自动根据商品成色调整价格

### 2. **通过二手平台与淘宝京东分析不同商品的历史价格变动**

比如说保值类：音响设备，十年来价格变动都很小

不保值类：手机电子产品

近期受市场波动价格变化大的：显卡（ai），硬盘（两年前的狂潮），



以及这些商品受商品成色的影响

同种商品（同内核同根），但是不同品牌的价格差距和贬值趋势

国内外同种商品的价格差距



- **基于多平台数据挖掘的商品价格波动分析与预测**
- **二手平台与电商平台商品价格历史数据对比研究**
- **商品价格趋势预测：基于闲鱼、淘宝、京东的时间序列分析**
- **不同类别的商品价格差异及其影响因素的数据分析**

**数据分析方向：**

- 收集闲鱼的二手历史价格数据，淘宝、京东的全新商品历史价格数据，进行数据清洗和预处理。
- 使用时间序列分析（如ARIMA、LSTM）预测价格趋势。
- 对比不同平台的价格差异，分析影响因素（如商品类别、品牌、发布时间、促销活动）。
- 通过回归分析或相关性分析研究价格波动与外部因素（如季节、节日）的关系。
